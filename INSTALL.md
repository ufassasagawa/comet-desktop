# Comet ☄️ インストール＆使い方（Mac）

プレゼンや画面共有の上に、参加者のコメントが弾幕で流れるアプリです。3分で入ります。

## インストール（初回だけ）

1. **`Comet-0.1.0.dmg` をダブルクリック**して開く
2. 出てきたウィンドウで **☄️ Comet を「Applications」フォルダにドラッグ**
3. アプリケーションフォルダの **Comet をダブルクリック**で開こうとする
   - 「Apple では確認できないため開けません」と出る → **「OK」** を押して閉じる（ここはまだ開けません）
4. **アップルメニュー  → システム設定 → 「プライバシーとセキュリティ」** を開く
5. 下までスクロールすると **「"Comet" は…ブロックされました」** と表示されているので、右の **「このまま開く」** をクリック
   - Touch ID かパスワードで承認 → 確認ダイアログでもう一度 **「このまま開く」**
6. これで起動します。**次回からは普通にダブルクリックでOK** 🎉

> ⚠️ 初回だけこの手順が必要です（Apple の安全確認を通していない社内配布アプリのため）。macOS の新しいバージョンでは「右クリック→開く」では開けず、上の**システム設定からの許可**が必要です。

起動すると最初にログイン画面が出ます → **Google でログイン**してください。

## イベントでの使い方（主催者）

当日の流れはこれだけです。

1. **ルームを作る** — 右上の **「＋ ルームを作る」** → イベント名（例：6月全社会議）を入れて **「作成する」**
2. **参加者URLを配る** — できたルームカードの **「コピー」** ボタンでURLをコピーし、Zoom/Meet のチャットに貼る（下の[案内文テンプレ](#参加者への案内コピペ用)が便利）
3. **弾幕を開く** — ルームカードの **「☄️ 弾幕を開く」** を押すと、透明な弾幕ウィンドウが画面に流れ始めます
4. **画面共有する** — Zoom/Meet では共有対象に **「ウィンドウ」ではなく「画面全体」** を選ぶ（ウィンドウ単体共有では弾幕が映りません）
5. 弾幕の表示/非表示は、メニューバー右上の **☄️ アイコン**、または **⌘⇧X** で切替できます

> ⚠️ **スライドは全画面（フルスクリーン）モードにしないでください。**
> 弾幕はデスクトップ画面の上に重なる仕組みのため、Keynote/PowerPoint 等を全画面モードにすると（macOS では別画面扱いになり）弾幕が流れません。スライドは**ウィンドウ表示のまま大きく**して発表するのがおすすめです。

### 表示先ディスプレイを選ぶ

外部モニターやプロジェクターに映す場合、弾幕を出す画面を選べます。

- メニューバーの **☄️ → 「表示先ディスプレイ」** から選択
- 既定は**外部ディスプレイ優先の自動選択**なので、通常はそのままでOK

### イベントのあとで

- ルームカードの **「ログ」**（終了後は「コメントログを見る」）から、流れたコメントを後で見返せます
- ルームURLは**発行から1週間で自動的に失効**します。イベントごとに新しいルームを作ってください

## 参加者への案内（コピペ用）

会議チャットにそのまま貼れる文面です。`{ここにURL}` を貼り付けたURLに置き換えてください。

```
📣 コメントを弾幕で流せます！スマホからどうぞ ☄️
{ここにURL}

・ログイン不要・名前（ニックネーム）は任意です
・コメントを書いて送ると画面に流れます（文字色も選べます）
・スタンプ（❤️👍🎉👏😂😮😢🤔）は連打OK！
```

## 困ったら

- **「開けません」「壊れているため開けません」と出る** → インストール手順4〜5の **システム設定 → プライバシーとセキュリティ → 「このまま開く」** で許可する
- **どうしても開けない（上級者向け）** → ターミナルで `xattr -dr com.apple.quarantine /Applications/Comet.app` を実行すると検疫フラグが外れて普通に開けます
- **Zoom/Meet で弾幕が映らない** → 画面共有は「ウィンドウ」ではなく **「画面全体」** を選ぶ
- **スライドを全画面表示にしたら弾幕が消えた** → 全画面モードは macOS では別画面扱いになるため弾幕が重なりません。スライドを**ウィンドウ表示に戻して**発表する
- **弾幕が違う画面に出る** → メニューバー ☄️ →「表示先ディスプレイ」で出したい画面を選び直す
- **参加者が「このルームは終了しました」と言われる** → URLが1週間で失効した可能性大。新しいルームを作り直してURLを配る
- **コメントが流れない** → 弾幕が非表示になっていないか確認（☄️ または ⌘⇧X で表示）。参加者が正しいルームURLを開いているかも確認
- **終了したい** → メニューバーの ☄️ →「終了」（または ⌘Q）。ダッシュボード右上の「アプリを終了」でもOK
